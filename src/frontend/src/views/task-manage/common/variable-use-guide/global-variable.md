# 说明
即，在作业中用户自定义的全局通用变量

# 用法

  使用 dollar 符 + 大括号：`${变量名}`

# 在哪里用

  - 字符串

    1. 脚本中直接引用，当前仅支持 `shell`
    2. “文件分发”步骤的源 `文件路径` 和 `目标路径`
    3. “执行脚本”步骤的 `脚本参数`

  - 命名空间

    脚本中直接引用，当前仅支持 `shell`

  - 主机列表

    无法直接引用，需要配合 “魔法变量”

  - 密文

    脚本中直接引用，当前仅支持 `shell`

  - 数组

    脚本中直接引用，当前仅支持 `shell`
